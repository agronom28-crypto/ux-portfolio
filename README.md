# UX Designer Portfolio | Sergey Nikolaenko

Personal UX/UI designer portfolio website with dark theme, scroll animations, and responsive layout.

## Features

- Dark premium theme with gradient accents
- Scroll-triggered fade-in animations
- Animated stat counters
- Responsive design (mobile, tablet, desktop)
- Pure HTML/CSS/JS, no frameworks
- CSS Grid & Flexbox layouts
- Smooth scrolling navigation
- Mobile hamburger menu
- Project cards with hover effects
- Skills, process, testimonials sections

## Tech Stack

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox, Animations)
- Vanilla JavaScript (IntersectionObserver API)
- Google Fonts (Inter)

## Sections

1. **Hero** - Introduction with animated stats
2. **Services** - UX research, UI design, prototyping, testing, MVP, design systems
3. **Projects** - Fintech Dashboard, Health App, E-commerce, SaaS CRM
4. **Process** - Research > Design > Build > Test
5. **Skills** - Tools, UX methods, technologies, product
6. **Testimonials** - Client reviews
7. **Contact** - CTA with Telegram & Email
